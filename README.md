# FCC-Apply-discount

This is the code I wrote that made me pass the Apply Discount challenge on FreeCodeCamp.org's Python bootcamp. 

What it does.
It uses a singular function with two parameters (price, discount). Within the function it will calculate the discount reduced total. The code will not run if neither your price nor discount are valid numbers,
your price is equal to or less than 0, or if your discount is less than 0 or greater than 100.

What I learnt and put to practice:
I learnt the saying that will stick with me: return is for code, print is for eyes. I learned the difference between return and print, how to validate types using isinstance(), and how to read a specification carefully; because code does exactly what you tell it, nothing more.

My struggle:
It took debugging, three rewrites and an invisible tabs vs spaces mystery to get here, but I got here.
